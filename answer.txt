1)What is a gitignore file and what should it include for a Java project?

2)What does fast-forward in git mean?
A) Fast forward means one branch is ahead of other branch in commits and the changes have to be committed to the branch.

3)Why should you have different branches? 
A)Branches are important in Git. Branches in git allows us to work on different user stories with different contexts.
